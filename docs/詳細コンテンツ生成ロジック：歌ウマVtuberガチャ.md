# **Vtuberコンテンツ生成ロジック：詳細仕様**

ガチャで排出されたVtuberの魅力を深掘りするための、4つの主要要素の実装アプローチです。

## **1\. チャンネル説明欄のLLM要約 (Gemini 3 Flash)**

高速な Gemini 3 Flash を活用し、説明欄からエッセンスを抽出します。

* **入力データ**: channels.list (snippet.description)  
* **プロンプト指針**:  
  * 「3行程度のキャッチーな紹介文にして」  
  * 「初見のリスナーが興味を持つようなポジティブなトーンで」  
  * 「歌の活動に関する記述を優先的に抽出して」

## **2\. 各種リンクの自動抽出**

説明欄に含まれるSNS（X, FANBOX, 公式HP等）のURLを抽出・整形します。

* **実装**: snippet.description から URL を抽出し、リンク集としてリスト表示。  
* **メリット**: ユーザーが即座に「推し」のフォローに行ける導線を作ります。

## **3\. 音声解析による「声の特徴」解説 (Gemini 2.5 Flash Native Audio)**

Gemini 2.5 Flash Native Audio Dialog のマルチモーダル機能を活用します。

* **プロセス**:  
  1. YouTube APIで取得した音楽動画（MV/Cover）の音声データをコンテキストとして入力。  
  2. **音声解析依頼**:  
     * 「この歌声を解析し、『高音の伸び』『ハスキーさ』『ビブラートの特徴』『似ている楽器や雰囲気』を詳しく分析して」  
* **出力**: 「透き通るようなクリスタルボイス」等のタグと、詳細な分析テキスト。

## **4\. サビメドレーの自動特定・ビジュアル再生 (Gemini 2.5 Flash Native Audio)**

音声対話・解析能力を活かし、楽曲の「サビ」をピンポイントで特定し、ライブ感のある演出で再生します。

* **技術的アプローチ**:  
  1. **サビ位置の特定**: Gemini 2.5 Flash に「この音声ファイルの中で、最も盛り上がるサビの開始時間と終了時間を mm:ss 形式で特定して」と依頼。  
  2. **再生制御**: YouTube IFrame APIを使用。特定された秒数を start / end パラメータに渡し、複数動画を順次再生。  
  3. **視覚演出（円形オーディオスペクトラム）**:  
     * Web Audio APIを用いて再生中の音声から周波数データをリアルタイム取得。  
     * **円形配置**: スペクトラムの始端と終端を丸く繋ぎ合わせ、チャンネルアイコンを囲むように円形に描画。  
     * **レイヤー構成**: チャンネルアイコンを中央に配置し、その外周で歌声に合わせて波形が躍動するデザイン。  
* **メリット**: 視覚と聴覚が同期することで、ユーザーは瞬時にそのVtuberの歌唱力のピークとキャラクター性を体感できます。

## **🌐 多言語対応（日英切り替え）戦略**

サイトの国際展開を見据え、以下のフローでコンテンツを生成します。

1. **ベース生成**: 上記 1, 3 のプロセスを日本語で行い、深いニュアンスを言語化。  
2. **AI翻訳**: 生成された紹介文・分析文を、Gemini 3 Flash を用いて英訳。  
   * **プロンプト**: 「以下のVtuber紹介文を、英語圏のリスナーがワクワクするような自然な英語に翻訳して。専門用語（歌枠など）は適切な英語表現（Singing Stream等）に置き換えて」  
3. **DB保存**: 日本語版と英語版の両方を Supabase の gacha\_history に保存。  
4. **UI切り替え**: フロントエンドの言語設定（i18n）に応じて表示を出し分け。

## **💡 実装上のアドバイス**

* **ネイティブオーディオの強み**: Gemini 2.5 Flash Native Audio は、単なるテキスト変換ではなく音の質感を直接理解するため、「サビの盛り上がり」の判定精度が非常に高いです。  
* **視覚効果の実装**: 円形のスペクトラムは Canvas API または SVG を使用し、requestAnimationFrame で描画することで滑らかな動きを実現できます。  
* **翻訳の品質**: 翻訳時には「Vtuber文化特有の言い回し」を反映させるためのコンテキストを Gemini 3 Flash に与えるのがコツです。
